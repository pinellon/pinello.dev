# pinello.dev

Site profissional para oferta de criação de websites, landing pages, lojas virtuais e sistemas web sob medida.

## Deploy

Este repositório está preparado para GitHub Pages usando GitHub Actions.

URL esperada:

```text
https://pinellon.github.io/pinello.dev/
```

## Estrutura

- `index.html`: página estática principal.
- `.github/workflows/deploy-pages.yml`: workflow de publicação no GitHub Pages.
- `.nojekyll`: evita processamento pelo Jekyll.
